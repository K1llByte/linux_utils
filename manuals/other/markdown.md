# Markdown

Markdown is a simple markup language for text formating


## Index



## Basic Syntax
```md
<!-- Headers -->
# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6

<!-- Modifiers -->
**Bold** or __Bold__
*Italic* or _Italic_
<ins>Underline</ins>
~~Strikethrough~~

<!-- Items List -->
- Item 1
- Item 2
- Item 3

<!-- Horizontal Separator -->
___

<!-- Line break -->
<!-- Two spaces after text (trailing whitespace) -->
Text1  
Text2
<!-- or -->
Text1<br/>
Text1

<!-- Tables -->
| Tables   |      Are      |  Cool |
|----------|:-------------:|------:|
| col 1 is |  left-aligned | $1600 |
| col 2 is |    centered   |   $12 |
| col 3 is | right-aligned |    $1 |

<!-- Blockquotes -->
> ##### This is a blockquote
>
> This is the next paragraph of the blockquote
> - Topic 1
> - Topic 2
>
>  *Everything* is going according to **plan**.

```

## HTML Extensions

```md
<!-- Colored text -->
<span style="color:#FF00FF">Pink</span>

<!-- Horizontal divided blocks -->
<div>
    <div style="float: left; width: 50%;">
        Text1
    </div>
    <div style="float: left; width: 50%;">
        Text2
    </div>
</div>

<!-- Small characters in superior and inferior opsitions -->
e<sup>-1</sup>
```
